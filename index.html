Learning the <pre> Element

What is the <pre> Element?

The <pre> element is used to display preformatted text. This means that the text will be shown exactly as it is written in the HTML file, including all whitespace and line breaks. 

The <pre> element:
- Preserves spacing and line breaks.
- Displays text in a monospaced font by default.
- Useful for code blocks, ASCII art, or aligned text.
